# recyclePOS
回收場POS機功能

# 功能

1. 編輯各品項名稱、價格
2. 各品項公斤數計算
3. 生產單一訂單收據
4. 產生日報表

![image](https://github.com/user-attachments/assets/8facf60b-2666-41a2-af42-40c8edd50069)
